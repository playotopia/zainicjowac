Packages Documentation

A package contains features to fit the various needs of your project. This can include any core features of Unity which is installed along with the Editor, or other packages which you can install as needed. Click on any of the following currently available packages to see the documentation for that package.

Released packages

The following packages have been tested with this version of Unity. To install these packages in your Project, go to Window > Package Manager.

  • Ads (preinstalled in every Project)
  • Analytics (preinstalled in every Project)
  • Package Manager UI (preinstalled in every Project)
  • In App Purchasing (preinstalled in every Project)
  • TextMesh Pro (preinstalled in every Project)
  • Asset Bundle Browser
  • Cinemachine
  • Oculus (Android)
  • Oculus (Desktop)
  • OpenVR
  • Post-processing
  • ProBuilder
  • Windows Mixed Reality
  • Xiaomi
