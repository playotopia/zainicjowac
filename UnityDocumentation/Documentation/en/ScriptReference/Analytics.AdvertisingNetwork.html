Description

The list of advertising or mediation network service providers.

Use the members of this list when possible to promote consistency in your Analytics data. However, if the advertising network is not included in this list, pass the argument to the Standard Event function as a string.

See Also: AnalyticsEvent.AdComplete, AnalyticsEvent.AdOffer, AnalyticsEvent.AdSkip, AnalyticsEvent.AdStart, AnalyticsEvent.PostAdAction.
